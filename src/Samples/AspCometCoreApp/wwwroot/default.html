Welcome to the .NET Core 'Chat' Web App

    This is a direct steal of code from the older "Chat-vs2010" web application.
    The older application relies on the DLL generated from the "AspComet-vs2010" library.
    This application relies on the DLL generated from the "AspCoreCometware" library.

The comet URL in the older app was /comet, whereas this app uses /comet/45.0,
as my goal was to mimick a SalesForce server (aka "org")

To change the comet path, look at the way the UseCometMiddleware is referenced in Startup.cs.
You may need to update the javascript source for the below client UIs, too.

Comet Terminal

Go to the Comet Terminal

Dojo

You can see when others join the 'chat' channel - launch multiple Dojo clients (multiple browser windows)

Go to the Dojo to start a chat session.
